  • Read the material safety data sheet (MSDS) and technical data sheet (TDS) for the resins that you're using design and fabricate a 3D mold (~ft2) and produce a fiber composite part in it

Prototype for Sitting table by Composite


Process

  • In this assignment we have used laser cutting sheet which combine and create a structure to support and act as form work.
  • Laser cutting the scaled version of table. Its Circular based ring which connects step wise to vertical support to form a structure.
  • Design is created using Rhino Software in 2D
  • After that it is laser cut and those part are assemble
  • Thin Plastic layer which will act as support ( Formwork ) for Composite
  • Use gloves for protection
  • Applying Resing ( Araldite Standard Epoxy Adhesive ) properly all side in thin layer
  • Covering it with Natural fiber sheet , here Jute Fiber sheet is used
  • Press fiber proper with lees force so it get properly layered with surface
  • After that apply one more layer of epoxy resin
  • Put whole assembly in vaccum bag for vacum process.
  • lock Vaccum bag properly so any air input should not happen while doing vaccum
  • Now turn on vaccum to get proper suction inside bag
  • Remove bag after composite get hard enough
